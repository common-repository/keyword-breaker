Import the Keyword Breaker data for your niche into your WordPress admin dashboard.


== Description ==
Import the Keyword Breaker data for your niche into your WordPress admin dashboard. This plugin will save the Keyword Breaker questions and keyword research as drafts ready to be written and published on your site.


== Installation ==
Download the zip file.
Log into WordPress, hover over Plugins, and click Add New.
Click on the Upload Plugin button.
Select the zip file you downloaded.
Click Install Plugin.
Activate.
